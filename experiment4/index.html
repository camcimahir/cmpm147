Description

Infinite worls is an exploration and building game. There are randomly generated island on the infinitly large map. The player can click on any water tiles to create a bridge tile and connect the islands.

Technical

I used procedural generation algorithms to create the infinite world. I used the XXH32 hashing function and p5.js methods such as the randomSeed method.
